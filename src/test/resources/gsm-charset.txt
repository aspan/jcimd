In Mobile telephony GSM 03.38 is a character set used in the
Short Message Service of GSM based cell phones. It is defined
in GSM recommendation 03.38. Messages sent via this encoding
can be encoded in the default GSM 7-bit alphabet, the 8-bit
data alphabet, and the 16-bit UTF-16 alphabet. Support of the
GSM 7-bit alphabet is mandatory for GSM handsets and network
elements, but characters in languages such as Arabic, Chinese,
Korean, Japanese or Cyrillic alphabet languages must be encoded
using the 16-bit UTF-16 character encoding or an extended
national language shift table.
